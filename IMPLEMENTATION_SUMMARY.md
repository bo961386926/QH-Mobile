## 🎯 功能详情

### 1. 设备比对模块 (device-comparison/)

#### device-selection.html - 设备选择页面
**主要特性：**
- 按地区分组显示设备列表（华东区、其他区域）
- 支持实时搜索设备名称和编号
- 显示设备状态（正常/故障中）
- 点击设备后保存到localStorage并跳转到比对页面

**设备样本数据：**
- DT-2024-009: 温度传感器·华东门区·装配线A (温感传感器)
- DT-2024-001: 温度传感器·华东门区·标准位置 (正常)
- DT-2024-002: 温度传感器·华东门区·分支位置 (正常)
- DT-2024-010: 温度传感器·华东门区·监测点 (故障中)
- DT-2024-011: 温度传感器·西区·A栋 (正常)

#### device-comparison.html - 设备比对数据采集页面
**主要特性：**

1. **比对类型选择**
   - 周期性校准（默认选中）
   - 故障排查
   - 验收测试

2. **标准设备值输入**
   - 显示最近更新时间：2024-06-20 14:25
   - 标准值输入框（预填充25.3°C）

3. **实测值输入**
   - 手动输入框
   - 蓝牙连接按钮（支持按钮显示）
   - 拍照OCR按钮（支持按钮显示）

4. **偏差自动计算**
   - 实时计算偏差值（实测值 - 标准值）
   - 显示偏差范围（±0.5°C）
   - 示例：标准值25.3，实测值26.1 → 偏差+0.8°C

5. **偏差原因记录**
   - 文本区域用于详细描述

6. **环境参数**
   - 下拉菜单选项：
     - 温度较高（>30°C）
     - 温度较低（<10°C）
     - 湿度较高（>80%）
     - 湿度较低（<30%）
     - 环境正常

7. **现场照片**
   - 三个标签页：手动输入、蓝牙连接、拍照OCR
   - 拖拽上传或点击拍照
   - 支持多张图片上传
   - 每张图片可单独删除

8. **底部操作**
   - 保存草稿按钮
   - 提交比对按钮

### 2. 设备更换模块 (device-replacement/)

#### device-replacement.html - 4步向导流程

**步骤指示器：**
1. 选择设备 (当前步骤)
2. 填写信息
3. 上传凭证
4. 确认

**第1步：选择设备**
- 信息提示：请选择要更换的设备
- 搜索框用于快速查找设备
- 设备列表与比对功能一致
- 支持单选，选中后显示复选框

**第2步：填写信息**
- 旧设备信息（自动填充）
  - 设备编号（只读）
  - 设备位置（只读）
  - 设备状态（可编辑：损坏/老化/维修等）
  
- 新设备信息（需填写）
  - 新设备编号（支持扫描条码）
  - 更换原因（必填）
  - 维护人员（可选）

**第3步：上传凭证**
- 旧设备照片
  - 点击拍照或上传照片
  - 显示已上传照片网格
  
- 新设备照片
  - 点击拍照或上传照片
  - 显示已上传照片网格

- 表单验证：至少需要上传一张照片

**第4步：确认**
- 显示旧设备信息汇总
- 显示新设备编号和更换原因
- 显示上传照片数量
- 确认无误后可提交

**向导功能：**
- 步骤指示器动态更新（显示已完成/当前/未开始）
- 上一步/下一步按钮
- 每步都有表单验证
- 提交前最终确认

## 🚀 使用流程

### 设备比对使用流程
1. 点击运维面板上的"比对"功能
2. 在设备列表中搜索并选择要比对的设备
3. 选择比对类型（周期性校准/故障排查/验收测试）
4. 输入标准值和实测值（系统自动计算偏差）
5. 填写偏差原因和环境参数
6. 上传现场照片（可选）
7. 点击"提交比对"完成

### 设备更换使用流程
1. 点击运维面板上的"设备更换"功能
2. 第1步：搜索并选择旧设备
3. 第2步：确认旧设备状态，输入新设备编号和更换原因
4. 第3步：上传旧设备和新设备的照片（必须至少一张）
5. 第4步：确认所有信息无误后提交
6. 系统确认提交完成
